This is a Small task Manger App.
It includes the following functionality
1.) Add New Task
2.) Edit the task
3.)Delete the task
4.)Search the Task
5.)Filter Option to see All, completed, pending Tasks
6.) Checkebox to check if the task is complete
7.) A percentage shows how much the task are completed

I have used teh following Hooks to complete these tasks
1.)useState
2.)useEffect
3.)useMemo
